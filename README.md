
# Viral Videos Tracker

เว็บแอพสำหรับติดตามวิดีโอที่กำลังเป็นไวรัลจากหลายแพลตฟอร์ม (YouTube, TikTok)

## คำอธิบาย
โปรเจกต์นี้ประกอบด้วย:
- backend: Node.js + Express API ที่ใช้ Puppeteer ในการ scrape หน้า Trending
- frontend: React + Vite สำหรับแสดงวิดีโอ
